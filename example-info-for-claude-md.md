# Example information for CLAUDE.md

## Overview

This is a small template to be used in a CLAUDE.md file to help
documentation assistants perform better. It helps the assistant gather context
faster and respect any project constraints and conventions. The information
here can of course be extended with further project-specific guidelines.

## Template

```
# Project Overview
Brief one-line project description clarifying the domain

## Project Structure
- **Primary language:**
- **Project type:**
- **Source code location:**
- **Tests location:**
- **Main branch:**

## Code Style & Conventions
- **Comment style:**
- **Documentation focus:**
  - Always document public APIs thoroughly
  - Document private methods only when complexity warrants explanation
```
